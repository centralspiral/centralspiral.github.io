SPIRAL-BOX

Phase I (ready to use)

Offers a dual approach to lossless data compression: The cloud-based API on AWS efficiently compresses data, reducing storage and transfer costs by up to 99%. For decompression, SPIRAL-BOX is required, allowing offline access to original data without internet. Alternatively, the SPIRAL-BOX standalone solution handles both compression and decompression in-house using built-in GPUs, eliminating the need for the cloud API and internet connectivity. This makes SPIRAL-BOX adaptable for small-scale environments with standard NAS storage and minimal infrastructure, yet capable of scaling up and integrating with cloud services for larger workflows. SPIRAL-BOX ensures efficient and cost-effective workflow management, whether operating locally or in the cloud.

SPIRAL-CACHE

Phase II & III (upcoming years)

Current microprocessors are facing increasing pressure in terms of memory capacity and effective bandwidth, particularly with the shift towards multi-core solutions. Amdahl's law, which dictates that computational performance requires proportional increases in memory capacity and effective bandwidth, has become a current bottleneck and a major issue. To address this challenge, SPIRAL-CACHE emerges as a solution by providing significantly improved effective bandwidth, benefiting both general-purpose processors and SoC accelerators in fields such as High-Performance Computing and Artificial intelligence.
